RAM

RAM (random-access memory) adalah memori jangka pendek di mana data disimpan sesuai kebutuhan prosesor. Ini tidak sama dengan data jangka panjang yang disimpan pada hard drive, yang tetap berada di sana walaupun komputer dimatikan

Jenis-jenis RAM

1. Static RAM (SRAM)
2. Synchronous Dynamic RAM (SDRAM)
3. Single Data Rate Synchronous Dynamic RAM (SDR SDRAM)
4. Double Data Rate Synchronous Dynamic RAM (DDR SDRAM)
5. Graphics Double Data Rate Synchronous Dynamic RAM (GDDR SDRAM)
6. Memori flash (flash memory)
7. Video RAM (VRAM)
